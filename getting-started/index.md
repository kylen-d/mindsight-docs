# Getting Started

MindSight is a multi-person gaze tracking and social phenomena analysis toolkit. This section walks you through installing the software, running your first analysis, and setting up a project for batch processing.

Pick a topic below to begin.

---

-   **[Installation](installation.md)**

-   **[Quickstart (CLI)](quickstart-cli.md)**

    Run the gaze pipeline from the command line on a single video.

-   **[Quickstart (GUI)](quickstart-gui.md)**

    Launch the PyQt6 interface and process a video with point-and-click controls.

-   **[Your First Project](first-project.md)**

    Create a project directory, configure a YAML pipeline, and batch-process multiple videos.
